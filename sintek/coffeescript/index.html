Coffeescript

Что это?

Язык программирования, построенный над Javascript

  • добавляет синтаксический сахар
  • добавляет отсутствующие возможности
  • предоставляет удобный доступ к хорошим частям Javascript
  • пытается улучшить плохие части Javascript

Как работает?

  • Любой Javascript является Coffescript
  • Coffescript транслируется в js перед использованием

Как установить и пользоваться?

  • npm install -g coffee-script
  • coffee --compile --output lib/ src/
  • codepen, jsfiddle, ...
  • Неявно, например в RoR

Сахар

небольшие приятности

Грамматика

  • конец строки заменяет ;
  • вместо фигурных скобок можно делать один отступ
  • скобки при передаче аргументов в вызов не обязательны (достаточно просто пробел)
  • функции задаются как () -> тело. Простейшая функция выглядит как ->
  • можно использовать зарезервированные слова как свойства объекта
  • return необязателен
